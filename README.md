# Biomedical spectra comparison module

## Setup

1. **Clone repository**
```bash
    git clone 
    cd 
```

2. **Virtual environment**
```bash
    python -m venv venv
    (or try python -m venv venv --without-pip)

    # Windows
    venv\Scripts\activate
    # Mac/Linux
    source venv/bin/activate
```

3. **Dependencies**
```bash
    pip install -r requirements.txt
```

4. **Run the app**
```bash
    python run.py
```

5. Open `http://127.0.0.1:5000`
